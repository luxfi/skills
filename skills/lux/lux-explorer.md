# Lux Explorer - Block Explorer

**Category**: Lux Ecosystem
**Related Skills**: `lux/lux-node.md`, `lux/lux-indexer.md`

## Overview

Lux Explorer is a **Blockscout-based block explorer** for browsing transactions, contracts, tokens, and network activity on Lux Network.

## Components

| Component | Path | Purpose |
|-----------|------|---------|
| explorer/ | ``github.com/luxfi/explorer`` | Blockscout backend |
| explore/ | ``github.com/luxfi/explore`` | Next.js frontend |

## Features

- Transaction search and details
- Contract verification and source code
- Token tracking (ERC20/721/1155)
- Network statistics and charts
- API for programmatic access

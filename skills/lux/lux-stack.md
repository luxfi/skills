# Lux Stack - Local Development Environment

**Category**: Lux Ecosystem
**Related Skills**: `lux/lux-node.md`, `lux/lux-universe.md`, `lux/lux-monitoring.md`

## Overview

Full local integration environment for Lux development — nodes, EVM, explorer, gateway, and monitoring via Docker Compose.

## Quick reference

| Item | Value |
|------|-------|
| Path | ``github.com/luxfi/stack`` |
| Stack | Docker Compose |
| Start | `docker compose up` or `make up` |

## Included Services

- Lux nodes (validators)
- EVM (C-Chain)
- Block explorer
- KrakenD API gateway
- Prometheus + Grafana monitoring
